Create a Vehicle hierarchy with a base Vehicle class and two subclasses: Car and Motorcycle. Implement a method getInfo() in the Vehicle class that returns a string with vehicle type and speed. Override getInfo() in subclasses to include additional information (number of doors for Car, has_sidecar for Motorcycle). The program should take a single integer input representing the speed and output the getInfo() result for both a Car and a Motorcycle instance with that speed.
Sample Input:
60

Sample Output:
Car: Speed 60 mph, Doors: 4
Motorcycle: Speed 60 mph, Sidecar: false


Sample Input:
80

Sample Output:
Car: Speed 80 mph, Doors: 4
Motorcycle: Speed 80 mph, Sidecar: false



Memory limit: 256 MB
Time limit: 5 seconds
