Write a program that reads an integer and a string from the input. If the integer is greater than 0, print the string that many times. If the integer is 0 or negative, print "Invalid input!" without throwing any exceptions.


Sample Input:
3
Hello

Sample Output:
Hello
Hello
Hello


Sample Input:
0
World

Sample Output:
Invalid input!



Memory limit: 256 MB
Time limit: 5 seconds
